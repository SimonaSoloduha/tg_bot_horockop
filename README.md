# Телеграм бот "Гороскоп" 

Перейти в тг: [Гороскоп от бабы Симы](https://tlgg.ru/@God_Willing_bot)

### Что умеет: 
* Гороскоп на сегодня
(Для всех знаков или для одного)
* Гороскоп на завтра
(Для всех знаков или для одного)
* Гороскоп совместимости для знаков
* Характеристики знаков зодиака
